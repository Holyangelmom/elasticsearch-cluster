`scp 10.194.186.44:/usr/local/elasticsearch /usr/local/elasticsearch`

```
node.name: node-2
node.master: false
node.data: true

http.port: 9200
http.enabled: false
```
3、修改elasticsearch文件夹权限  
chown -R elastic:elastic /usr/local/elasticsearch
